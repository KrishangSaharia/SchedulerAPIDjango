## Setup
To install all the dependencies run: 

```
$ pip install -r requirements.txt
```

## Run :
```
$ python manage.py makemigrations
$ python manage.py migrate
```

## To start the server:
```
$ python manage.py runserver
```
### Now , you can start deploying at your local host.
